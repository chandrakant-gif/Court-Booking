# QuickCourt 🏟️

QuickCourt is a full-stack web application that allows users to discover sports venues and book courts online easily. The platform provides a smooth booking experience with sport filtering, venue browsing, and secure payment integration.

---

## 🚀 Features

* 🏟️ Browse available sports venues
* 🔎 Filter courts by sport
* 📍 View venue details and location
* 📅 Real-time court booking
* 💳 Secure payments using Stripe
* 👤 User authentication
* 🖼️ Venue image gallery
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

### Frontend

* React.js

### Backend

* Node.js
* Express.js

* MongoDB

* Stripe API

## ⚙️ Installation

### 1. Clone the repository

```
git clone https://github.com/chandrakant-gif/Court-Booking.git
```

### 2. Navigate to project folder

```
cd Court-Booking
```

### 3. Install backend dependencies

```
cd server
npm install
```

### 4. Install frontend dependencies

```
cd ../client
npm install
```

---

## ▶️ Running the Application

### Start Backend

```
cd server
npm run dev
```

### Start Frontend

```
cd client
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server folder**.

Example:

```
PORT=5000
MONGO_URI=your_mongodb_connection
STRIPE_SECRET_KEY=your_stripe_secret
```
